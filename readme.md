# SkyPeek 天气小部件

一个使用 Python 和 PySide6 制作的桌面天气预报小部件。

## 特性

* 显示当前天气（温度、描述、湿度、风速）
* 未来几小时天气预报（占位符数据）
* 可拖动无边框窗口
* 圆角和半透明背景
* 极简模式切换
* 右键菜单（刷新、置顶、退出）
